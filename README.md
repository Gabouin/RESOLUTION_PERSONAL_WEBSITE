# RESOLUTION - GENERAL CODING PATH

8-weeks coding challenge for RESOLUTION - Hack Club.

**WEEK 1 - PERSONAL WEBSITE**

I always dreamed of a website that could be used as a portfolio of my projects ! This pathway was the good reason to start understanding how to make one. I firstly designed the basics like the text, links, images and other things using html, guided by the WEEK 1 guide. After that, I also followed the guide to make the CSS and searched code on the internet because I was stuck trying to do animations when you go in the white squares. I changed the colors, the fonts, the places...
